
# KIP Track

Student project app for KIP tracking. There is no real data involved.

KIP Track is an android mobile application developed exclusively as a Final Project (UAS) for The Mobile Device Programming II course. It serves as an educational simulation to demonstrate how KIP (Kartu Indonesia Pintar) Kuliah funds can be monitored and managed transparently.

This app simulates the synergy between University Administrators, Students, and Guardians to ensure accountability in the usage of educational funds.

### Important Disclaimer

This application is strictly for educational purposes!

Dummy Data: All data contained within the app (University Names, Student Profiles, Financial Balances, and Transaction History) is 100% FICTITIOUS (DUMMY)

No Real Money: This app DOES NOT process real money, connect to any banking systems, or facilitate actual financial transactions. It is a simulation environment only.

Project Scope: Built to demonstrate Android development skills using Firebase Authentication and Firestore.

### Key Features

For Students: 
- Dashboard Simulation: View simulated remaining balances and upcoming semester allowances.
- Expense Logging: Input daily expenses with categories (e.g., Books, Transport, Food).
- Proof Upload: Simulate uploading transaction receipts (using the device camera/gallery) for validation.
- Status Tracking: See if expenses are "Approved" or "Rejected" by the Admin.

For Guardians:
- Monitoring View: Read-only access to the connected student's spending history.
- Transparency: Verify that funds are being used for educational necessities.

For Administrators:
- Database Management: Simulate adding or removing Universities and Study Programs.
- User Management: Manage Student and Guardian accounts.
- Validation Logic: Review student expense logs and simulate the approval or rejection process based on compliance rules.

### Technical Highlights
- Firebase Integration: Demonstrates real-time data syncing using Cloud Firestore.
- Multi-User Roles: distinct interfaces and permissions for Admin, Student, and Wali.
- Modern UI: Built with Jetpack Compose for a clean, native Android experience.
